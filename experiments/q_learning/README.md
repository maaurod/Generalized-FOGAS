# Q-learning experiments

There is no standalone Q-learning experiment suite in this folder yet.

The reusable implementation lives in `src/rl_methods/q_learning`, and current
Q-learning usage appears as a supporting baseline in broader experiment scripts,
especially the MountainCar comparisons under `experiments/fogas/scripts`.

Use this folder for future isolated Q-learning notebooks or scripts if the
baseline needs to be studied independently from FOGAS, FQI, or SBEED.
